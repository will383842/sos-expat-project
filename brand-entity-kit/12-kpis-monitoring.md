## Red flags — actions correctives

| Symptôme | Diagnostic probable | Action |
|----------|---------------------|--------|
| Position `sos expat` stagne à 5-8 après 60j | Manque de presse / sources Wikidata | Relancer presse + doubler annuaires |
| GBP reste peu vu (<100 vues/mois) | Fiche incomplète ou catégories mal choisies | Audit fiche + compléter attributes |
| Wikidata item supprimé | Sources insuffisantes ou promotionnel | Attendre plus de presse, reformuler NPOV |
| 0 retombée presse malgré 130 envois | Pitch mal ciblé ou mal écrit | Refaire un A/B test pitch, cibler + précis |
| Avis Google négatifs | Qualité service / support | Audit expérience utilisateur (hors scope SEO) |
| Knowledge Panel partiel (pas de photo) | Logo Commons pas reconnu | Re-uploader logo Wikimedia Commons avec licence claire |

## Synthèse — ce qui fait bouger l'aiguille

Par ordre d'impact sur la position `sos expat` :

1. **Wikidata live + populated** : +2 à +4 positions sur brand
2. **GBP complet + 20+ avis** : +1 à +2 positions + Knowledge Panel visible
3. **3+ articles presse DR50+** : +1 à +2 positions + sources Wikidata
4. **30+ annuaires NAP cohérents** : +1 position + autorité générale
5. **Social posts réguliers + hashtags brand** : effet secondaire de reconnaissance

**Objectif réaliste 90 jours** : position `sos expat` de 9,54 → **2-3**
et **Knowledge Panel présent** sur Google.fr, Google.com, Google.be.
